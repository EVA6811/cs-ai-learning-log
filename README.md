# cs-ai-learning-log
A personal learning log for computer science, AI, and bootcamp studies.

## 목적
이 저장소는 부트캠프 수업 복습, Python/AI 기초 자습, 머신러닝·딥러닝 실습, LLM 에이전트 개발 준비 과정을 기록하기 위해 만들었습니다.

## 폴더 구성
- `00-roadmap`: 전체 학습 계획과 주간 회고
- `01-bootcamp`: 부트캠프 수업 복습 및 실습 기록
- `02-python-ai-basics`: Python, NumPy, pandas, matplotlib 기초 연습
- `03-data-structures-algorithms`: 자료구조와 알고리즘 기초
- `04-math-for-ai`: AI 학습을 위한 수학 정리
- `05-ml-dl-projects`: 머신러닝/딥러닝 미니 프로젝트
- `06-llm-agent`: LLM API, RAG, 에이전트 실습
- `07-paper-review`: 논문 요약, 개념 정리, 구현 기록

## 기록 방식
각 학습 내용은 다음 형식으로 정리합니다.

1. 오늘 배운 개념
2. 직접 작성한 코드
3. 헷갈렸던 부분
4. 오류와 해결 방법
5. 다음에 공부할 내용
