### Pandas
| 개념              | 의미                                    |
| --------------- | ------------------------------------- |
| DataFrame       | 엑셀의 시트처럼 행과 열로 이루어진 2차원 데이터 구조        |

# 실습
## 데이터 처리
1) 엑셀 파일 읽기
- pd.read_excel('filename.xlsx')
- pd.read_excel('filename.xlsx', skiprows=number) : 위에서 number 만큼의 행을 무시하고 읽기.
  - skiprows=[...] : 특정 행 번호만 제외 가능.
  - usecols=[...] : 필요한 column만 선택 가능.

2) 열 삭제  
- del df['' : '']

3) 행 삭제 
- df.drop(number)

*# inplace=True : 현재 DataFrame 자체를 수정한다. 예) df.drop(number, inplace=True)*

4) 컬럼명 바꾸기
- df.columns = [...] : 새 컬럼명의 개수가 기존 컬럼 개수와 같아야 함.
  - 예) 기존:[A,B,C] -> df.columns = [D,E,F]
- df.rename(columns={'' : ''})

5) index명 바꾸기
- df.set_index('')

6) 두 DataFrame 병합
- pd.merge(df1, df2, on='')
  - on='' : ''에 해당하는 칼럼이 같은 행끼리 합친다.

7) 새로운 열(columns) 만들기
- df[] = ''

8) loc와 iloc
- df.loc[...] : index 이름으로 행을 찾는 방식.
- df.iloc[...] : 숫자 위치로 행을 찾는 방식.

*[number:number] / ['index1':'index2']도 가능*

## 상관관계 분석
np.corrcoef(df[...], df[...])
### 해석 기준
| 상관계수 절댓값 | 해석         |
| -------- | ---------- |
| 0.1 이하   | 거의 관계 없음   |
| 0.3 이하   | 약한 상관관계    |
| 0.7 이하   | 뚜렷한 상관관계   |
| 1.0 이하   | 매우 높은 상관관계 |

## 시각화
df[...].sort_values(ascending=False).plot(kind='bar')
- ascending : True(오름차순) / False(내림차순)
- kind : 'bar':막대그래프, ...

### 로그 스케일
plt.yscale('log')
- 비교하는 두 값의 크기 차이가 너무 크면 같은 그래프에서 작은 값이 거의 보이지 않기 때문에, Y축을 로그 스케일로 바꾼다.

### 산포도(Scatter)
- s(size) : 마커(점)의 면적을 지정
  - s=df[...] : df[...]의 수치에 따라점의 크기를 다르게 만든다.
- c(color) : 마커(점)의 색상을 지정
  - c=df[...] : df[...]의 수치에 따라 점의 색상이 연속적으로 바뀐다.

#### 추세선
회귀 방정식 : np.poly1d(np.polyfit(x, y, 1))

- NaN/Inf 포함되면 오류 발생.
- f1 : 회귀 방정식으로 선언한 함수.

| 코드                     | 의미                             |
| ---------------------- | ------------------------------ |
| `np.polyfit(x, y, 1)`  | x와 y 관계를 가장 잘 설명하는 1차 함수 계수 계산 |
| `np.poly1d(fp1)`       | 계산된 계수로 실제 함수 생성               |
| `plt.plot(..., 'r--')` | 빨간 점선으로 경향선 표시                 |

### 항목 이름 표시
plt.text(x,y,'')

### 기타
- plt.xlabel() : x축 이름 설정
- plt.ylabel() : y축 이름 설정
- plt.colorbar() : 그래프 옆에 색상의 기준을 보여주는 막대 모양의 눈금자(컬러 바)를 추가하는 함수
- plt.grid() : 격자무늬 활성화
- plt.title('') : 시각화 내용 상단 이름 정하기
- plt.savefig('', dpi=200) : 시각화 내용 저장하기(png)
  - dpi : 그래프의 해상도(화질)를 결정하는 매개변수
- plt.show() : 시각화 내용 보여주기
- df.to_excel('filename.xlsx') : 수정한 데이터파일 저장.
