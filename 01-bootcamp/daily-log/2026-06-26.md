### 기타 내용
#### LiDAR와 Sensor Fusion
LiDAR는 카메라와 달리 거리 정보를 직접 얻을 수 있다.
| 센서      | 장점                       |
| ------- | ------------------------ |
| 카메라     | 색상, 텍스처, 표지판, 신호등 인식에 강함 |
| LiDAR   | 거리, 3D 공간 구조 파악에 강함      |
| Radar   | 악천후, 속도 추정에 강함           |
| GPS/IMU | 위치, 자세 추정에 사용            |

#### SLAM
- Simultaneous Localization And Mapping
- 내 위치를 추정하면서 동시에 지도를 만드는 기술

#### Corner
- 좋은 특징점
- 선 하나만 있으면 위치가 애매하지만, 코너는 위치가 비교적 명확하기 때문
#### SIFT
- Scale-Invariant Feature Transform
- 크기, 회전, 일부 가림이 있어도 반복적으로 찾을 수 있는 특징점을 추출하는 방법
- 실제 같은 물체도 다음처럼 다르게 보이기 때문에 이용함.

| 변화    | 예시              |
| ----- | --------------- |
| 거리 변화 | 멀리 있으면 작게 보임    |
| 회전    | 카메라 각도에 따라 기울어짐 |
| 가림    | 일부가 안 보임        |
| 조명 변화 | 밝기와 색이 달라짐      |

#### Classification
어떤 클래스인지 분류하는 작업.
#### Segmentation
- 이미지의 각 영역 또는 픽셀을 의미 단위로 나누는 작업.
- 자동차 비전에서는 Detection보다 Segmentation이 더 정밀할 수 있다. (bounding box로는 지나갈 수 있는 공간까지 막힌 것으로 판단될 수 있기 때문)
