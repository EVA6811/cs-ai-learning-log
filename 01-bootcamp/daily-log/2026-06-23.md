## 본 내용

### 1. AI, 머신러닝, 딥러닝
이 개념들을 너무 물리적인 실체처럼 생각하지 말고, 논리적 포함 관계로 이해하라는 것.
AI ⊃ Machine Learning ⊃ Deep Learning

### 2. 컴퓨터 비전

| 수준                | 내용               | 예시                       |
| ----------------- | ---------------- | ------------------------ |
| Low-level Vision  | 이미지 자체를 개선하거나 변환 | 흐림 제거, 선명화, 색 보정, 노이즈 제거 |
| Mid-level Vision  | 특징, 경계, 영역을 찾음   | 엣지 검출, 특징점 추출, 세그멘테이션    |
| High-level Vision | 의미를 판단함          | 이미지 분류, 객체 탐지, 행동 인식     |

### 3. 필터와 노이즈
필터는 이미지에서 원하는 성분은 강조하고, 원하지 않는 성분은 줄이는 처리.

| 노이즈                   | 설명                     |
| --------------------- | ---------------------- |
| Salt and Pepper Noise | 흰 점, 검은 점이 랜덤하게 생기는 잡음 |
| Impulse Noise         | 특정 픽셀 값이 갑자기 튀는 잡음     |
| Gaussian Noise        | 전체적으로 밝기값이 흔들리는 잡음     |

노이즈가 많으면 모델이 객체를 제대로 인식하기 어렵다.

### 4. 평균 필터와 가우시안 필터
#### 평균 필터
주변 픽셀값의 평균을 내서 현재 픽셀값을 바꾸는 방식입니다.

장점 : 이즈가 줄어든다는 것.
단점 : 이미지가 흐려진다는 것.


#### 가우시안 필터
가까운 픽셀에 더 큰 가중치를 주고, 먼 픽셀에는 작은 가중치를 주는 방식.

평균 필터보다 자연스럽게 부드럽게 만드는 데 자주 쓰임.

### 5. 샤프닝 필터
이미지를 선명하게 만드는 처리

원본 이미지 - 부드럽게 만든 이미지 = 디테일/경계 정보

물체 경계가 더 선명해질 수 있다.

### 6. 엣지와 엣지 검출
엣지는 이미지에서 밝기나 색이 급격하게 바뀌는 부분. 
쉽게 말하면 경계선이다.

| 기준값 | 결과                       |
| --- | ------------------------ |
| 낮음  | 작은 차이도 엣지로 판단 → 선이 많이 나옴 |
| 높음  | 큰 차이만 엣지로 판단 → 선이 적게 나옴  |

배경과 객체 색이 비슷하면 엣지를 찾기 어렵다.

### 7. 업샘플링과 보간
이미지를 크게 만들 때 단순히 픽셀을 복사하면 이미지가 깨져 보임.
그래서 중간값을 추정해야 함.
이 과정을 보간이라 한다.

| 개념   | 설명                         |
| ---- | -------------------------- |
| 업샘플링 | 작은 이미지를 크게 만드는 것           |
| 보간   | 새로 생긴 빈 픽셀값을 주변 값으로 추정하는 것 |

### 8. Classification
이미지를 여러 클래스 중 하나로 판단.

픽셀 숫자 배열 → 특징 추출 → 클래스 확률 계산 → 가장 높은 클래스 선택

### 9. Decision Boundary
분류 모델은 입력 공간을 여러 영역으로 나눈다.
이때 클래스가 갈라지는 경계를 Decision Boundary, 즉 결정 경계라고 한다.

### 10. Feature
| 특징    | 설명                |
| ----- | ----------------- |
| 색상    | RGB 값, 색상 히스토그램   |
| 모양    | 외곽선, 형태           |
| 질감    | 표면 패턴             |
| 경계    | 물체와 배경의 구분        |
| 위치 관계 | 특징들이 어디에 배치되어 있는지 |

중요한 개념은 "불변성"이다.
#### 불변성
좋은 특징은 변화가 있어도 잘 유지된다.

| 변화           | 설명                   |
| ------------ | -------------------- |
| 위치 변화        | 물체가 화면의 다른 곳에 있어도 인식 |
| 크기 변화        | 가까이 있거나 멀리 있어도 인식    |
| 회전 변화        | 방향이 바뀌어도 인식          |
| 가림 Occlusion | 일부가 가려져도 인식          |

### 11. ANN, CNN
#### ANN
인공신경망. 입력값을 받아 여러 층의 계산을 거쳐 결과를 내는 구조.

Input Layer → Hidden Layer → Output Layer

#### CNN
합성곱 신경망. 이미지에 필터를 적용하면서 특징을 추출하고, 그 특징을 바탕으로 분류나 탐지를 수행한다.

이미지 입력 → Convolution → Sampling/Pooling → Feature 추출 → Classification

### 12. Object Detection
이미지 안에서 어디에 무엇이 있는지 찾는다.

| 작업               | 질문                     |
| ---------------- | ---------------------- |
| Classification   | 이 사진은 무엇인가?            |
| Localization     | 그 물체가 어디에 있는가?         |
| Object Detection | 여러 물체가 각각 어디에 있고 무엇인가? |
| Segmentation     | 픽셀 단위로 어느 영역이 무엇인가?    |

YOLO는 대표적인 객체 탐지 모델이다.

### 13. Segmentation
이미지를 영역별로 나누는 작업이다.

| 종류                    | 설명                    |
| --------------------- | --------------------- |
| Semantic Segmentation | 같은 클래스는 모두 같은 영역으로 표시 |
| Instance Segmentation | 같은 클래스라도 개체별로 따로 구분   |

### 14. 객체 탐지가 어려운 이유
| 어려움    | 설명                   |
| ------ | -------------------- |
| 다양한 모양 | 의자도 종류가 매우 많음        |
| 크기 변화  | 가까운 물체와 먼 물체의 크기가 다름 |
| 회전     | 방향이 바뀜               |
| 조명     | 밝기 변화로 특징이 달라짐       |
| 가림     | 일부가 안 보임             |
| 배경 혼동  | 배경과 객체가 비슷함          |

많은 데이터를 학습해야 하고, 좋은 라벨링이 중요하다.
