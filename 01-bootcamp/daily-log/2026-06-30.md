## 본 내용
### 불필요한 열 삭제
del은 컬럼 삭제에 자주 쓰이고, 실행하면 원본 DataFrame에서 바로 삭제된다.

### 컬럼명 정리
        CRIME.columns = [
            '구별',
            '살인발생', '살인검거',
            '강도발생', '강도검거',
            '강간강제추행발생', '강간강제추행검거',
            '절도발생', '절도검거',
            '폭력발생', '폭력검거'
        ]

*새 컬럼명 리스트의 개수와 기존 컬럼 개수가 반드시 같아야 한다*

### 합계 행 삭제
        CRIME.drop(0, inplace=True)
- drop(0)은 0번 index 행을 삭제한다는 뜻.
- inplace=True는 결과를 새 DataFrame으로 반환하지 않고, CRIME 자체에 바로 반영하겠다는 뜻.

### '-'값을 0으로 바꾸기
- 값이 없는 경우 -로 표시된 곳이 있다. '-'는 문자열 이므로 연산이 불가능하다.

      CRIME = CRIME.replace('-', 0)
*inplace=True가 가능하지만 공식적으로 지양하는 방법이다.*

### 검거율이 100%를 넘는 이유
- 2024년에 검거된 사건이 반드시 2024년에 발생한 사건만은 아니기 때문.
- 이전 연도에 발생한 사건이 2024년에 검거될 수 있다.
- 때문에 100% 초과 값을 100으로 제한하는 방향으로 진행함.

      df.loc[df[...] > 100, ''] = 100
- 조건을 만족하는 행만 선택해서 값을 바꾸는 Boolean Indexing 예시로 [조건, 선택하는 행] 이다.]

### 결측치(NaN) 처리
- NaN은 Not a Number, 즉 계산 불가능한 값이다.
- 해석상 “발생한 사건이 없었으므로 미검거가 발생한 것도 아니다”라는 관점에서 100으로 채웠다.
- df.fillna()를 이용해 특정 값으로 채울 수 있다.

### 발생이라는 글자 제거
    for crime in CRIME.columns:
      if crime.endswith('발생'):
        CRIME.rename(columns={crime: crime[:-2]}, inplace=True) 

| 코드                      | 의미                  |
| ----------------------- | ------------------- |
| `endswith('발생')`        | 문자열이 `발생`으로 끝나는지 확인 |
| `crime[:-2]`            | 문자열 끝의 두 글자 제거      |
| `rename(columns={...})` | 컬럼명 변경              |

### 데이터 정규화(Normalization)
- 절도 100건과 살인 1건을 단순히 같은 단위로 더하면 범죄의 상대적 중요도나 규모 차이가 왜곡될 수 있다.
- 그래서 각 범죄 컬럼을 0~1 사이 값으로 바꾼다.

#### MinMaxScaler
    from sklearn import preprocessing
    x = CRIME[crime_index]
    min_max_scaler = preprocessing.MinMaxScaler()
    x_scaled = min_max_scaler.fit_transform(x.astype(float))
  
각 컬럼의 최솟값을 0, 최댓값을 1로 바꾸고, 나머지 값은 그 사이 비율로 변환.

*MinMaxScaler가 이상치에 약할 수 있다.*

### NumPy 배열을 다시 DataFrame으로 만들기 
MinMaxScaler를 적용하면 결과가 Pandas DataFrame이 아니라 NumPy 배열로 나온다.

    CRIME_norm = pd.DataFrame(
        x_scaled,
        columns=x.columns,
        index=CRIME['구별']
    )

*컬럼명과 각 구 이름 index가 유지가 된다.*

### axis
| 코드       | 의미                |
| -------- | ----------------- |
| `axis=0` | 행 방향으로 계산, 즉 열별 합 |
| `axis=1` | 열 방향으로 계산, 즉 행별 합 |

df.loc[:,:].sum(axis=1) : 특정 행과 열의 범위의 각 요소들의 합을 열 방향으로 계산한다.

    df1['A','B','C'] = df2['A','B','C']

df2의 세 열의 내용을 d1에 같은 이름으로 합친다.

### Seaborn Pairplot
Pairplot : 여러 변수들 사이의 관계를 한 번에 보여주는 그래프.

    sns.pairplot(
        df[
            ['A', 'B', 'C', 'D', 'E', 'F']
        ]
    )
    plt.show()

#### hue
각 점이 어느 요소인지 구분하기 위해 hue=''을 사용한다.

    sns.pairplot(
        CRIME_norm[
            ['A', 'B', 'C', 'D', 'E', 'F']
        ].reset_index(),
        hue=''
    )
    plt.show()

*reset_index()는 index로 들어가 있던 구별을 다시 일반 컬럼으로 빼내는 역할을 한다.*

#### kind='reg'
회귀 경향선을 같이 보고 싶으면 kind='reg'를 사용한다.

    sns.pairplot(
        CRIME_norm[
            ['A', 'B', 'C', 'D', 'E', 'F']
        ],
        kind='reg'
    )
    plt.show()

*각 산포도에 회귀선과 신뢰구간이 함께 표시된다.*

### Heatmap
Heatmap : 2차원 데이터를 색상으로 표현하는 그래프

plt.figure(figsize=(6, 12), dpi=100)

    sns.heatmap(
        CRIME_norm[
            ['A', 'B', 'C', 'D', 'E', 'F']
        ],
        annot=True,
        fmt='.2f'
    )
    plt.show()

| 옵션           | 의미             |
| ------------ | -------------- |
| `annot=True` | 칸 안에 숫자 표시     |
| `fmt='.2f'`  | 소수점 둘째 자리까지 표시 |
| `cmap`       | 색상표 설정         |

*어느 요소의 어떤 수치가 높거나 낮은지 한눈에 볼 수 있다.*

#### Color Map
heatmap의 색상표를 바꿀 수 있다.

- YlGnBu : 노랑->초록->파랑
- coolwarm : 파랑->빨강
- coolwarm_r : 빨강->파

*_r은 reverse, 즉 색상 순서를 뒤집는다는 뜻*

### Folium 지도 시각화
Folium : 지도 시각화 라이브러리

    import folium
    map_name = folium.Map(
        location=[x, y],
        zoom_start=15
    )

- location : 중심 좌표
- zoom_start : 확대 정도

#### Marker

    folium.Marker(
        [x, y],
        popup=''
    ).add_to(map_name)

#### CircleMarker

    folium.CircleMarker(
        [x, y],
        radius=50,
        popup='',
        fill_color='#ff0000'
    ).add_to(map_name)

*위치와 함께 크기를 이용해 값을 표현할 때 유용함.*

### Choropleth 지도
Choropleth : 지역별 값을 색상으로 칠하는 지도

*seoul_geo.json은 서울시 각 구의 이름과 경계 좌표를 포함하고 있으므로, Folium이 이를 이용해 구역별 색칠을 할 수 있다.*

    seoul_map = folium.Map(
        location=[x, y],
        zoom_start=11
    )
    crime_map_data = df.reset_index()[['A', 'B']]
    folium.Choropleth(
        geo_data=seoul_geo,
        data=crime_map_data,
        columns=['A', 'B'],
        key_on='feature.properties.name',
        fill_color='PuRd'
    ).add_to(seoul_map)

#### key_on='feature.properties.name'
지도 파일의 properties.name 값과 데이터프레임의 구별 값을 서로 맞춰서 색칠한다.
